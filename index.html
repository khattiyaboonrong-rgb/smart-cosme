คำต้องห้ามตาม พ.ร.บ. เครื่องสำอาง พ.ศ. 2558

มาตรา 41 — ระบบตรวจจับอัตโนมัติ

คำเหล่านี้ถูกตรวจจับโดยอัตโนมัติ ทั้งในช่องกรอกข้อมูล (Tab ออกแบบฉลาก) และการอ่านจากรูปภาพ (Tab ตรวจจากรูป)
📌 มาตรา 41(1) — โฆษณาเกินจริง / เป็นเท็จ
ที่สุด / ที่สุดในโลก ปลอดภัย / ปลอดภัย 100% ไร้สารพิษ / ไร้สารตกค้าง ไม่ระคายเคืองผิว ไม่มีผลข้างเคียง จบทุกปัญหาผิว / จบทุกปัญหาสิว ขาวทันที / เห็นผลทันที ผ่านการรับรองจาก อย. ขาวไวกว่าฉีด หน้าเรียวไม่ง้อโบท็อกซ์ เต่งตึงเหมือนยิงเลเซอร์
📌 มาตรา 41 วรรคสอง(2) — ก่อให้เกิดความเข้าใจผิดในสาระสำคัญ
ดูดซับสารพิษ / ล้างสารพิษ กระชับสัดส่วน สลายไขมัน ลดความอ้วน / ลดน้ำหนัก ต่อต้านอนุมูลอิสระ กระตุ้นการสร้างคอลลาเจน ลดการสร้างเม็ดสีผิว ลดผมหงอก เร่งการงอกของเส้นผม เผาผลาญไขมัน
📌 มาตรา 41 วรรคสอง(3) — แสดงสรรพคุณเป็นการรักษาโรค
รักษา / รักษาสิว / รักษาโรค หายขาด กำจัดสิวถาวร ป้องกันหรือรักษาฝ้า ระงับ/ยับยั้งแบคทีเรีย ยับยั้งเชื้อรา ฆ่าเชื้อ ต้านมะเร็ง ลดการอักเสบของผิวหนัง ทาผิวกายเพื่อไล่ยุง/แมลง โกลมเส้นผมกำจัดเหา บรรเทาอาการปวด รักษาแผล โรคผิวหนัง / โรคปลาริกินต์
📌 มาตรา 41 วรรคสอง(4) — สรรพคุณบำรุงกาย (เกินขอบเขตเครื่องสำอาง)
แข็งตัวได้นานขึ้น เพิ่มขนาดและความแข็งแรง กระชับภายใน / ช่องคลอดฟิต บรรเทาช่องคลอดแห้ง เพิ่มสมรรถภาพทางเพศ ชะลอการหลั่ง แก้ปัญหาไม่แข็งตัว ชะลอความชรา รีแพร์ เสริมสมรรถภาพ
⚖️ อ้างอิง: พ.ร.บ. เครื่องสำอาง พ.ศ. 2558 | คู่มือโฆษณาเครื่องสำอาง ฉบับปรับปรุง พ.ศ. 2567 สำนักงาน อย.
